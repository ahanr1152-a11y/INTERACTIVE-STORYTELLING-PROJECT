# INTERACTIVE-STORYTELLING-PROJECT
# 📖 StoryVerse – Interactive Storytelling Website

An interactive storytelling web application where users can choose different adventures and experience engaging, choice-based stories through a modern and visually appealing interface.

---

# ✨ Features

- 🌳 Choose from multiple story adventures
- 📖 Interactive storytelling experience
- 🎨 Modern and attractive UI
- 🌌 Beautiful immersive background
- ⚡ Smooth animations and transitions
- 📱 Responsive design for different screen sizes
- 🖱️ Easy-to-use navigation
- 💡 Clean and beginner-friendly code structure

---

# 🛠️ Built With

- HTML5
- CSS3
- JavaScript

---

StoryVerse/

# 🚀 Getting Started

1. Clone this repository.
2. Open the project folder.
3. Run `index.html` in your browser or use VS Code Live Server.
